Origami
Butterflies
These are two butterfly designs from the book "Origami Butterflies" by Michael LaFosse, folded out of sticky notes.
Modular dodecahedron
This is a dodecahedron built using 30 modules, with 10 sticky notes for each of the three colours. Note that each vertex has all three colours: blue, purple, and pink.
Modular icosahedron
This is a modular icosahedron built using 30 modules. Each module uses half of a sticky note.
